# GitHub-Profile-Finder


A simple app to search GitHub users and display their profile info.

## Features
- Search by username
- Show avatar, name, bio, followers, following, repos
- Link to GitHub profile

## How to Run
1. Clone the repo
2. Open `index.html` in your browser
3. Enter a GitHub username and click Search

## Deployment
- Works on CodePen (paste HTML, CSS, JS)
- Works on GitHub Pages (upload repo and enable Pages)
